RUMUS Mean

Dalam statistika dan matematika, mean disebut juga dengan rata-rata. Jadi, definisi mean adalah nilai rata-rata dari sekumpulan data yang terdiri dari dua atau lebih.

RUMUS Mean untuk data tunggal $\(\bar{x}=\frac{\sum x_i}{n} \)$ Keterangan:

\(\bar{x}:\) nilai rata-rata

\(\mathrm{x}_{\mathrm{i}}\) : nilai data ke-i

n : banyaknya data

RUMUS MEDIAN

Median disebut juga dengan nilai tengah. Median diperoleh dengan cara mengambil nilai tengah pada data yang telah disusun secara berurutan dari angka terkecil ke terbesar.

Pada data tunggal untuk menentukan median yaitu: Misalkan banyaknya data n,

Untuk \(n\) ganjil: \(M e=x_{\frac{1}{2}(n+1)}\)

Untuk \(n\) genap: \(M e=\frac{\frac{x_n}{2}+\frac{x_n}{2}+1}{2}\)

Keterangan:

\(x_{\frac{n}{2}}=\) data pada urutan ke- \(\frac{n}{2}\) setelah diurutkan.

  • Standar Deviasi

Standar deviasi atau simpangan baku biasanya diajarkan pada ilmu statistik untuk mengukur tingkat kesamaan atau kedekatan dalam suatu kelompok.

Standar deviasi adalah nilai statistik yang sering kali dipakai dalam menentukan kedekatan sebaran data yang ada di dalam sampel dan seberapa dekat titik data individu dengan mean atau rata-rata nilai dari sampel itu sendiri.

RUMUS STANDAR DEVIASI

Untuk Data Tunggal : \(S=\sqrt{\frac{\sum\left(x_i-\bar{x}\right)^2}{n}}\)

\(S=\) Standar deviasi

\(x_i=\) nilai \(\mathrm{x} \mathrm {ke-i}\)

\(\bar{x}=\) nilai rata-rata data

\(n=\) jumlah data

Persentil dalam statistik dan statistika istilah persentil kerapkali dipergunakan untuk memahami dan menginterpretasikan data, yang menunjukkan nilai di bawah yang persentase tertentu dari suatu kumpulan data yang ada.

RUMUS PERSENTIL DATA TUNGGAL

\[ P_i=\text { data } k e-\frac{i(n+1)}{100} \]

Jika persentil 25% maka: $\( P_i=\text { data } k e-\frac{25(n+1)}{100} \)$

Jika persentil 50% maka: $\( P_i=\text { data } k e-\frac{50(n+1)}{100} \)$

Jika persentil 75% maka: $\( P_i=\text { data } k e-\frac{75(n+1)}{100} \)$

Keterangan:

\(\mathrm{P}=\) persentil ke \(-\mathrm{i}\)

\[ i=1,2,3, \ldots, 99 \]

n = banyak data
