  • Lakukan proses diskritisasi dengan equal width dan equal frequency

  • Kumpilkan tugas dengan link github (web statis dari jupyter book)

1. Pengertian#

–> Diskritisasi yaitu proses mengubah data numerik menjadi data kategorikal

2. Implementasi#

–> Diskritisasi dengan Equal Width dan Equal Frequency

  • Equal width : dikritisasi yang membagi data numerik menjadi beberapa kelompok dengan lebar kelompok yang kurang lebih sama besar.

  • Equal frequency : dikritisasi yang membagi data numerik menjadi beberapa kelompok dengan jumlah anggota perkelompok yang kurang lebih sama besar.

–> Diskritisasi dengan basis Entropy

Entropy digunakan untuk menentukan yang manakah node yang akan menjadi pemecah data latih berikutnya

3. Lakukan proses diskritisasi dengan basis entropy#

Entropy adalah

  • Rumus Entropy:

\[Entropy (D_{1}) = -\sum_{i=1}^{m} Pi \log_{2} Pi\]

  • Rumus Gain:

\[Gain(E_{new}) = E_{initial} - E_{new}\]

  • Rumus Info:

\[Info_{A}(D) = \frac{|D_{_{1}}|}{|D|} Entropy(D_{1}) + \frac{|D_{2}|}{|D|} Entropy(D_{2})\]

Definisi discretization#

  • Proses mengubah data bertipe data numeric menjadi data bertipe categorical

  • Diskritisasi memiliki 2 pendekatan yaitu equal-width intervals dan equal-frequency intervals

  • equal-width intervals adalah diskritisasi dengan lebar data yang sama

  • equal-frequency intervals adalah diskritisasi dengan jumlah data yang sama

ENTROPY#

Diskritisasi berbasis entropi (entropy based discretization) adalah salah satu jenis algoritma diskritisasi terawasi yang menggunakan mekanisme top- down. Tujuan dari algoritma ini adalah mendapatkan partisi yang mengandung baris data dari kelas yang sama sebanyak mungkin.

  • Rumus Entropy:

\[Entropy (D_{1}) = -\sum_{i=1}^{m} Pi \log_{2} Pi\]

  • Rumus Gain:

\[Gain(E_{new}) = E_{initial} - E_{new}\]

  • Rumus Info:

\[Info_{A}(D) = \frac{|D_{_{1}}|}{|D|} Entropy(D_{1}) + \frac{|D_{2}|}{|D|} Entropy(D_{2})\]

Dalam membuat fungsi GetOverollCategory (keseluruhan) yang digunakan untuk menghitung data keseluruhan yang nantinya digunakan untuk menghitung entropy

def GetOverollCategory(col):
  group = sample.loc[:, :].groupby("category").count()
  a = group.loc["A", col]
  b = group.loc["B", col]
  return (a,b,a+b)

Dalam fungsi splitter digunakan untuk membuat split antara value yang telah ditentukan lalu mengembalikan data yang telah dipisahkan

def splitter(value, col):
  #get data less and greater from value
  less = sample[sample[col] <= value]
  greater = sample[sample[col] > value]

  #calculate into category for each data
  less_group = less.loc[:, :].groupby("category").count()
  greater_group = greater.loc[:, :].groupby("category").count()

  #get value based on category
  less_category_A = less_group.loc["A",col]
  less_category_B = less_group.loc["B",col]
  greater_category_A = greater_group.loc["A",col]
  greater_category_B = greater_group.loc["B",col]

  return (
      [less_category_A, less_category_B, less_category_A + less_category_B],
      [greater_category_A, greater_category_B, greater_category_A + greater_category_B]
  )

Buat fungsi entropy untuk mencari nilai entropy

def entropy (d):
  r1 = (d[0] / d[2]) * np.log2(d[0] / d[2])
  r2 = (d[1] / d[2]) * np.log2(d[1] / d[2])
  return np.sum([r1, r2]) * -1

Membuat fungsi info dan gain

def info(d):
  r1 = (d[0][2] / sample.shape[0]) * entropy(d[0])
  r2 = (d[1][2] / sample.shape[0]) * entropy(d[1])
  return r1 + r2

Fungsi gain untuk menghitung selisih antara entropy awal dengan yang baru

def gain(Einitial, Enew):
  return Einitial - Enew
